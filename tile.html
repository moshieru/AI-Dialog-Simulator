Отклонения партнёров

Табличный срез по ключевым показателям.
Итоговая оценка партнёра считается по текущему периоду: красная карточка -1 балл, жёлтая 0 баллов, зелёная +1 балл.

Красная зона 0 score меньше 0, без новичков
Желтая зона 0 score от 0 до 4, без новичков
Зеленая зона 0 score от 5 и выше, без новичков
Новички 0 Не участвуют в расчете процентов
Главное отклонение 0% —
текущий месяц выполнен
есть прирост или была норма в прошлом месяце
текущий месяц не выполнен без положительной динамики
